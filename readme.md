# Installation

### 1) Install
````
$ composer require alex-oliveira/ao-scrud
````

### 2) Configure "config/app.php" file
````
'providers' => [
    /*
     * Vendor Service Providers...
     */
    AoScrud\ServiceProvider::class,
],
````
